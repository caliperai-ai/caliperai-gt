# 2D Polygon Tool

The **Polygon Tool** creates freeform polygon shapes on camera images.

## When to Use

- Irregular shaped objects (not rectangular)
- Precise segmentation boundaries
- Complex object outlines

## Activating the Tool

- **Keyboard**: Press `P` or `3`
- **Toolbar**: Click polygon icon

## Creating a Polygon

### Click Method
1. Click to place first vertex
2. Click to place additional vertices
3. Double-click or press Enter to close polygon

### Drawing Method
1. Click and drag to draw freeform
2. Points are automatically placed along path
3. Release and close with double-click

## Editing Polygons

### Move Vertices
- Click and drag individual vertices

### Add Vertices
- Click on edge to add new point

### Delete Vertices
- Select vertex and press Delete
- Or right-click → Delete Point

### Move Entire Shape
- Drag from inside polygon

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| P or 3 | Select polygon tool |
| Enter | Close polygon |
| Escape | Cancel current |
| Delete | Delete selected vertex |
| Backspace | Remove last vertex |

## Tips

1. **Start with corners**: Place key points at sharp turns
2. **Follow edges**: Outline should follow object boundary
3. **Fewer vertices**: Use minimum points needed
4. **Smooth curves**: Place points closer together for curves
5. **Check zoom**: Verify accuracy at full resolution
