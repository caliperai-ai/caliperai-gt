# Keyboard Shortcuts

## General

| Key | Action |
|-----|--------|
| ? | Show keyboard shortcuts help |
| Ctrl+S | Save |
| Ctrl+Z | Undo |
| Ctrl+Shift+Z | Redo |
| Escape | Cancel current action / Deselect |
| Delete | Delete selected annotation |

## Tool Selection

| Key | Action |
|-----|--------|
| B or 1 | 3D Box tool |
| R or 2 | 2D Rectangle tool |
| P or 3 | Polygon tool |
| V | Selection tool |
| S | AI Segment tool |

## View Navigation

| Key | Action |
|-----|--------|
| 1-4 | Switch view presets |
| Home | Reset camera view |
| + / - | Zoom in/out |
| Arrow keys | Pan view |
| Space + drag | Pan view (hold) |

## Frame Navigation

| Key | Action |
|-----|--------|
| ← | Previous frame |
| → | Next frame |
| Shift+← | Jump 10 frames back |
| Shift+→ | Jump 10 frames forward |
| Home | First frame |
| End | Last frame |
| Space | Play/Pause |

## Annotation Editing

| Key | Action |
|-----|--------|
| Enter | Confirm annotation |
| Escape | Cancel |
| Delete | Delete selected |
| [ | Rotate left 5° |
| ] | Rotate right 5° |
| Shift+[ | Rotate left 1° |
| Shift+] | Rotate right 1° |

## Tracking

| Key | Action |
|-----|--------|
| T | Link selected boxes to track |
| P | Propagate track |
| I | Interpolate track |
| G | Go to track start |
| Shift+G | Go to track end |

## Selection

| Key | Action |
|-----|--------|
| Click | Select single |
| Ctrl+Click | Add to selection |
| Shift+Click | Range select |
| A | Select all (current frame) |
| Ctrl+A | Select all visible |

## Quick Actions

| Key | Action |
|-----|--------|
| F | Focus on selected |
| Double-click | Zoom to object |
| C | Copy annotation |
| V | Paste annotation |
| H | Hide selected |

## Tips

1. **Learn shortcuts gradually** - Start with basic navigation
2. **Use cheat sheet** - Press `?` anytime
3. **Customize** - Some shortcuts can be remapped
